## Unity Roll-a-Ball

A simple 3D game created in Unity by following the Unity Learn Roll-a-Ball tutorial.

## Features

Player movement using physics
Collectible pickups
Score tracking
Win condition UI
Simple obstacle and enemy behavior

## Built With
Unity
C#
Universal Render Pipeline (URP)

## How to Play

Use the keyboard controls (WASD) to move the player around the map and collect all pickups.

## How to Run

In Unity:

Open the project using Unity Hub
Open the main scene
Press Play in the Unity Editor

Build Version:

Run the executable file inside the Builds folder.
